# Lambda-Calculus Interpreter 1.0.0

A pure F# library for parsing, interpreting, and transpiling lambda-calculus expressions, including support for church numerals, beta reduction, and basic file imports.

---

## Features

- Lambda expressions, variables, and applications.
- Evaluates expressions in either normal or applicative mode.
- Church encoding and delta reduction.
- Top-level variable assignments and simple imports.
- Transpiles lambda calculus programs into string representations.
